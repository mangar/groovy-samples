Groovy-Samples by mangar

This is a sample project used to improve my skills on Groovy until I am reading the book: Groovy in Action.
Occasionally I shall use others books as reference: Groovy Recipes, Programming Groovy and Begining Groovy and Grails

Book Covers





Samples

Until now, the samples are covering:

  • XML
  • String /GString
  • Closure
  • Regular Expressions
  • Control Structures
  • Patterns
  • Multimethods
  • Spread
  • Mix-in
  • Filters
  • IO
  • Multithread
  • Process
  • Templates
  • Groovylets (Groovy + Servlets)
  • Database access


Dependencies

Some libraries are necessaries for compile and run the project:

  • Groovy 1.5.7 - download groovy-all-1.5.7.jar
  • JUnit - download junit.jar version 4
Whereas you are using Eclipse IDE, I recommend you to also use the Groovy Eclipse plugin it can save some time for you.
Groovylets sample I am using Jetty as the Servlet container.
For database access you'll need HSQLDB library
